# Telecommunication Company Network 📡

A **CCNA-level** project designing a secure enterprise network for a telecommunications company, built and simulated in **Cisco Packet Tracer**. It combines VLAN segmentation, OSPF routing, VoIP, wireless access, and an ASA firewall with a DMZ.

---

## 🎯 Project Goal

Build a realistic, secure corporate network for a telecom company — supporting voice and data traffic, wireless clients, and a protected public-facing zone (DMZ) behind a firewall.

---

## 🧩 Key Features

- **VLANs** — segmented traffic for departments, voice, and management
- **OSPF** — dynamic routing across the network
- **VoIP** — voice traffic support with a dedicated voice VLAN
- **Wi-Fi** — wireless access for client devices
- **ASA firewall + DMZ** — perimeter security with a demilitarized zone for public-facing servers

1. Open the `.pkt` file in **Cisco Packet Tracer**.
